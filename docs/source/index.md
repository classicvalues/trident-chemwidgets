# trident_chemwidgets

Version: {{ release }}

Chemical widget set to work in jupyter notebook and lab.

## Quickstart

To get started with trident_chemwidgets, install with pip:

```
pip install trident_chemwidgets
```

or with conda:

```
conda install trident_chemwidgets
```
